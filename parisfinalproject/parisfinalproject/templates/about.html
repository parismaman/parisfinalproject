About the project

The project is part of our computer science studies at the 10th grade in Tichonet high school.

The project is part of our final Bagrut and part of the study phase this year in data science class.

The project combines all the elements we learned this year and presents them on one web page that contains several pages

The elements we used in the project are:

  • Flask
  • wtf
  • Pandas
  • Bootstrap

In the project, I located databases on Kaggle.com and downloaded them into the computer. I analyzed them and presented them in an easy way that when every person sees that he can easily understand and draw conclusions himself.

Through the project, I learned how to learn on my own and find information online on my own

The end result of the project is an available website that is pleasing to the eye and easy to understand for any user
